## OCR 粗扫后提炼出的高价值词系

> 338 张图跑完 OCR 后，真正值得保留的不是所有文本，而是反复出现的“人格壳子词”。

### 1. 群名 / 头衔上下文

- `蓝色裤头猫`
- `LV100无头`
- `拓麻歌子`
- `电子宠物`
- `歌姬`

这些词在截图里反复出现，但更像当时的群名片、头衔和显示名，不应该直接当成猫头人格本体。
它们更适合拿来判断语境和截图来源，而不是当成稳定口头禅。

### 2. 高频自我叙事

- `我是匹诺曹，姓匹不是很正常`
- `不是我吹`
- `我一拳可以倒八个免爷`
- `由我来承受所有的罪责，所有的歧视`
- `我是二刺猿`

这组文本很关键：猫头会在“吹牛 / 发癫 / 自我献祭 / 自黑 / 二刺猿身份认领”之间快速跳。

### 3. 高频关系模板

- `欢迎参加我和X的甜蜜婚礼`
- `免爷想和我结婚想疯了吧`
- `叫我老公`
- `留在群里的老光棍只会越来越`

这说明图梗里的猫头经常把关系话题直接推到：

- 婚礼
- 结婚
- 老公
- 光棍

也就是一种“抽象婚恋模板”。

### 4. 高频下三路模板

- `给我浦西`
- `嘬你牛牛`
- `我要贷款割包皮`
- `我会把屁股插进你的几把`
- `卖屁股￥20一次`
- `我的勾八给你`

OCR 证实这些不是几张孤例，而是一整条高频词带。

### 5. 高频评价模板

- `真变态，猫头都接受不了`
- `猫头也是致癌物`
- `tm猫头原来这么恶心啊粉转黑了爬`
- `胡说！我到底哪里像变态了`
- `不过有一说一我真的不变态`

这里能看到经典张力：

- 一边被人叫变态、恶心、致癌物
- 一边自己又辩解“不变态”

### 6. 高频对象模板

- `肖战肉棒`
- `草神 / 散兵`
- `免爷`

这些对象本身未必重要，重要的是：猫头味总是通过“把一个具体人/角色拖进自己那套黑话系统”来完成。
